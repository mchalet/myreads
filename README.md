# My Reads - Udacity Project #1
Built with `create-react-app`.

## Getting Started
1. Run `git clone https://github.com/mchalet/myreads.git` to clone this repository.
2. Move to this new directory with `cd myreads`.
3. `npm install` 
4. `npm start`
